## **1️⃣ نصب پکیج‌ها**

```bash
npm install
```

## **2️⃣ ساخت دیتابیس در pgAdmin**

راست کلیک روی **Databases → Create → Database** → نام: **auth_system**

## **3️⃣ ویرایش `.env`**

(**DB_PASSWORD** رو بذار)

## **4️⃣ تست اتصال**

```bash
npm run test:db
```

## **5️⃣ ساخت ادمین**

```bash
npm run init:admin
```

## **6️⃣ اجرای سرور**

```bash
npm run dev
```
